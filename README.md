本仓库用于归档和发布多个独立的 HTML 页面。这些页面本身不关联任何特定网站或域名，仅作为静态资源存储和分发使用。

## 项目简介

`guide-panel-7mscores` 是一个静态 HTML 页面集合仓库。页面内容以独立文件形式存放，便于通过 GitHub Pages 或其他静态托管服务进行访问。仓库本身不提供导航、搜索或动态功能，所有页面均为自包含的 HTML 文件。

## 目录说明

```
.
├── index.html          # 仓库首页（可选）
├── page1.html          # 示例页面
├── page2.html          # 示例页面
└── assets/             # 公共资源（如 CSS、JS、图片等）
```
